Instalação

Rode em segundos

Clone o repositório e execute o instalador correspondente ao seu sistema.

# Linux / macOS
git clone https://github.com/miguel9w/IAS-CANVAS-TOOL.git
cd IAS-CANVAS-TOOL
chmod +x install.sh && ./install.sh
# Windows (PowerShell)
git clone https://github.com/miguel9w/IAS-CANVAS-TOOL.git
cd IAS-CANVAS-TOOL
powershell -ExecutionPolicy Bypass -File .\install.ps1

Depois de instalar, inicie o Core:

cd core-platform && npm run dev
Funcionalidades

O que você pode fazer

Widgets em tempo real

A IA gera componentes React com JSX que são compilados e renderizados no navegador instantaneamente — sem rebuild.

Canvas infinito

Pan com o botão do meio do mouse. As janelas flutuam em coordenadas absolutas dentro de uma camada sem limites.

Múltiplos agentes

Suporte nativo a OpenCode e Pi Coding Agent. Cada um com seu plugin específico, mesma API de widgets.

Janelas flutuantes

Arraste, redimensione e feche janelas como em um desktop. Z-index dinâmico — a última interagida fica por cima.

Babel Standalone

JSX é compilado no próprio navegador. Widgets podem usar import do React e hooks como useState.

Reconexão automática

O WebSocket reconecta automaticamente se o plugin cair. Status visível no canto superior direito do canvas.

Exemplo

Widget de contador

Peça à IA algo como "crie um widget de contador no canvas". O código abaixo é o que o widget recebe como source_code:

function Widget({ appBus }) {
  const [count, setCount] = React.useState(0);
  return (
    <div style={{ padding: 24, textAlign: 'center' }}>
      <h2>Contador: {count}</h2>
      <button onClick={() => setCount(c => c + 1)}>+</button>
      <button onClick={() => setCount(c => c - 1)}>-</button>
    </div>
  );
}
